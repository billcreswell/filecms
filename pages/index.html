Experimental File Based Website.

Main Idea: File based website to allow VIP to maintain a website.

Use

The VIP would be able to open (s)ftp folders and either drag new files in, or edit existing, with the tool of their choice

Files

  • Main app, "/app/excite.php" finds files in specified folders, of either html or txt, and displays them with a standard template
  • There is a php file search at search.php
